A simple plugin to fetch and display daily quotes using a public API and shortcode.

== Description ==

This plugin allows you to insert a motivational quote using `[daily_quote]`. It fetches content from an open API and displays it with a simple, styled layout.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin from the Plugins screen
3. Use `[daily_quote]` in your content
